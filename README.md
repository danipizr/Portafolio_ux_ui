Daniel Pizarro — UX/UI Portfolio

Repositorio que contiene la construcción y desarrollo de mi portafolio UX/UI, orientado a presentar proyectos, proceso de diseño y enfoque profesional en experiencias digitales centradas en el usuario.
________________________________________
Propósito
Este proyecto tiene como objetivo estructurar y desarrollar un portafolio web desde cero, aplicando principios de:
•	Diseño centrado en el usuario
•	Arquitectura de la información
•	Responsive design
•	Buenas prácticas de desarrollo front-end
El enfoque no es solo visual, sino también estructural y técnico.

Descripción de componentes
index.html
Página principal del portafolio. Contiene: - Hero section - Sección “Sobre mí” - Proyectos - Proceso de diseño - Formulario de contacto
________________________________________

styles/
base.css
Define la estructura visual del sitio: - Layout - Tipografía - Grid y flexbox - Estilos generales
animacion.css
Contiene: - Animaciones - Transiciones - Efectos visuales (fade, scroll, etc.)
________________________________________

js/
main.js
Gestiona la interactividad del sitio: - Menú móvil - Animaciones al hacer scroll - Eventos de navegación
________________________________________

image/
Contiene recursos gráficos utilizados en el portafolio: - Imagen de perfil (DanielPizarro.png)
________________________________________

cases/
Contiene los casos de estudio:
proyecto-calma.html
Caso de estudio UX completo que incluye: - Contexto - Problema - Proceso - Resultados
b_fica_page
Archivo adicional asociado a proyectos o exploraciones de diseño.
